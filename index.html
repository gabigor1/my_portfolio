Web Development Projects

Space Invaders

A simplified version of Space Invaders, created using HTML5, CSS, vanilla JavaScript, game logic and audio effects. A hard task primarily due to a 7-day delivery period.

The Lockdown Chef

A pair project with a 48 hours delivery time. The goal was to develop an effective app to provide a list of food recipes. The user types the ingredients they want to use for cooking and a list of recipes appear. The app was delivered using React.js, Express, Axios and online APIs for the database (Spoonacular).

Spot.me

Working within a team of 4 members, the goal was to create a full-stack social app to find gyms, post workouts and share feedback and suggestions. The front-end was developed using React.js with third-party packages (Mapbox) and APIs from Google to conduct research and map gym locations. The back-end was built using self developed models and databases. I worked mainly styling the app, used Bulma and SASS; I also worked with the front-end of the app, feedBack page and Profile, making sure the users receive all the data of the messages and posts.

Film-Space

A simple full-stack app where users can add films and share reviews. I created the back-end environment using Django with Python. The front end was delivered using React.js, Express, Axios and a third-party package (Star-rating).
